Goal

Learn basic HTML, CSS and GitHub for beginners... using iPad Pro

Initial set-up

  1. Download GitHub and Working Copy apps
  2. Create GitHub account
  3. In GitHub create a personal site (GitHub Pages)
    • Clone repository template (provided by Scotty)
    • Remember to rename clone using your exact account name
  4. Import repository into Working Copy
  5. Create a file and start coding in HTML and CSS!
